

This "Fotag(y35wen)" android mobile application is created by Yang Wen (20608279)


- Platform: 
	MacOS High Sierra version 10.13.1
	android studio Version 3.1.3


- Version of Java: 
	10.0.1



- Control:
 	* Use the "load" icon to load ten images on screen.
	* Press clear button will clear the loaded images also the rateFilter in the toolbar.
	* Support custom filtering images by clicking the "stars".
	* User can clear each image loaded rating by clicker the reset button next to the image rating bar.
	* After clearing the images, user can still reloaded them by clicker load button, and everything is restarted.
	
	
	
- Note:
	* After clicking the load button or clear button, the program might need few seconds to respond, so please wait for few seconds.
